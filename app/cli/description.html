Application Description & Documentation

About the Application

Sabates Vidal is an online platform that allows company sales representatives to register and manage customer orders. This first version focuses on the creation, storage and visualization of orders.

Functionalities

  • 1. Presentation: Home page with logo and navigation buttons.
  • 2. Description: Explains all functionalities.
  • 3. Operations Menu: Allows the user to select actions.
  • 4. Create Order: Fill in a form to create and store new orders.
  • 5. View All Orders: Shows all stored orders with prices including VAT.
  • 6. View One Order: Search and display a specific order by its code.
